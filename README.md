## 📖 项目简介

本仓库为宠物科普公众号提供**高质量、系统化**的内容支持。所有文章力求**科学严谨、通俗易懂、有温度**，适合养宠新手和资深铲屎官阅读。

## 💡 为什么选择这个项目

| 优势 | 说明 |
|:---:|------|
| 🔬 **科学严谨** | 引用兽医学和动物行为学权威知识 |
| 📖 **通俗易懂** | 专业概念生活化表达，配比喻辅助理解 |
| 💛 **有温度** | 科普中融入对动物的关爱，不是冷冰冰的知识堆砌 |
| 📐 **标准模板** | 统一格式，便于排版和批量生产 |
| 🐾 **覆盖全面** | 猫、狗、异宠、健康四大领域，200 篇体系化内容 |

## ✍️ 写作风格指南

| 原则 | 说明 | 示例 |
|:---:|------|------|
| 🔬 **科学为本** | 引用可靠的兽医学和动物行为学知识，避免伪科学 | "根据《小动物内科学》第 5 版..." |
| 💬 **通俗表达** | 用生活化的语言解释专业概念，适当使用比喻 | "猫咪的呼噜声就像人类的微笑" |
| 💛 **情感温度** | 在科普中融入对动物的关爱 | "每一个毛孩子都值得被温柔对待" |
| 🎯 **实用导向** | 每篇文章都提供可操作的建议和注意事项 | "建议每 3 个月做一次体内外驱虫" |
| 📐 **结构清晰** | 统一使用标准模板，便于排版和阅读 | 段落分明，小标题引导 |
