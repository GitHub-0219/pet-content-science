## ✍️ 写作风格指南

1. **科学为本**：引用可靠的兽医学和动物行为学知识，避免伪科学
2. **通俗表达**：用生活化的语言解释专业概念，适当使用比喻
3. **情感温度**：在科普中融入对动物的关爱，让读者感受到温暖
4. **实用导向**：每篇文章都提供可操作的建议和注意事项
5. **结构清晰**：统一使用标准模板，便于排版和阅读
